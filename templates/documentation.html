Overview

The Educational Keylogger Demonstration is a Python-based tool designed to showcase how keyloggers function. It is specifically created for educational purposes to help security professionals, students, and educators understand the mechanics behind keyboard monitoring software.

This tool demonstrates:

  • Basic keyboard input capture
  • System information collection
  • Log file creation and management
  • USB deployment mechanisms

Installation

To use this educational demonstration, you'll need:

  • Python 3.6 or higher
  • Windows operating system
  • pynput library
# Clone the repository
git clone https://github.com/yourusername/educational-keylogger.git

# Navigate to the project directory
cd educational-keylogger

# Install dependencies
pip install -r requirements.txt

For USB deployment demonstration:

  • Format a USB drive (for demonstration purposes)
  • Run the USB package creator function
  • Copy the generated files to the USB drive

Usage

This tool is designed for educational purposes only and should be used in a controlled environment with proper authorization.

Basic Usage:

# Import the module
from keylogger import EducationalKeylogger

# Create an instance
keylogger = EducationalKeylogger(log_dir="logs", collect_system_info=True)

# Start monitoring (for educational demonstration)
keylogger.start()

Creating USB Demonstration Package:

# Import the function
from keylogger import create_usb_package

# Create the package
create_usb_package(output_dir="usb_demo")

Always ensure you have proper authorization before running this demonstration.

Features

  • Keyboard Monitoring: Captures keyboard inputs and logs them to a file.
  • System Information Collection: Gathers basic system information for educational purposes.
  • Log Management: Creates organized log files with timestamps.
  • USB Deployment: Demonstrates how keyloggers can be deployed via USB drives.
  • Educational Safeguards: Includes multiple consent requirements and disclaimers.

Technical Details

The Educational Keylogger Demonstration uses the following technical approaches:

Keyboard Monitoring:

The tool uses the pynput library to register keyboard event listeners. When a key is pressed, the on_press callback function is triggered, which processes and logs the keystroke.

System Information Collection:

Basic system information is collected using Python's standard libraries (platform, socket, os) to demonstrate how keyloggers might gather contextual information.

Log File Management:

Logs are stored in a designated directory with filenames based on dates and times. This demonstrates how captured data might be organized.

USB Deployment:

The USB deployment function creates the necessary files for running the demonstration from a USB drive, showing how malicious software might be distributed.

Ethical Considerations

Using keyloggers without explicit consent is illegal and unethical. This educational demonstration is provided to increase awareness and understanding of these tools for defensive purposes only.

Ethical guidelines when using this demonstration:

  • Only use on systems you own or have explicit permission to test
  • Inform all users that their keystrokes will be monitored during the demonstration
  • Do not use any captured data for any purpose other than education
  • Delete all logs after the educational demonstration is complete
  • Never deploy on public or shared computers without authorization

Legal Warning

Unauthorized use of keyloggers may violate various laws, including:

  • Computer Fraud and Abuse Act
  • Electronic Communications Privacy Act
  • State and national privacy laws
  • Workplace privacy regulations

The author of this educational tool assumes no liability for misuse.
